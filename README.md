# wow-ts

好的教程可以一路运行。
我们要做一个能够一路拷贝代码运行的ts教程。集成了许多优秀代码片段，在开发过程中可以随时复制。同时搭配了一个练手项目，一个纯前端，用来做时间管理的项目。ts+vue3+vite架构的。开源项目和开源教程可以打配合。开源项目里用到的知识写入开源教程。开源教程所用到的例子来源于开源项目。互相印证。
